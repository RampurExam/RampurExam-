Introduction of CSS

Cascading Style Sheets.
Cascading : refers to the procedure that determines which style will apply to a certain section, if you have more than one style rull.
Style : How you want a certain part of your pages to look.you can set things like color, margins, font, etc for things like tables, paragraphs, and headings.
Sheets: Sheets are templates, or a set of rules, for determining how the webpage will looks.
The first version of CSS was invented 1996 by Hakon Wium Lie.

Selector

ELEMENT Selector : The element selector select HTML element based on the element name.
p{Property:value}
CLASS Selector :To select the element with a particul of class,use(.)
character with specifying the class name.
.class_name{Property:value}
ID Selector :The id attribute is the unique identifier in an HTML document.
The id Selector is used with a # character.
